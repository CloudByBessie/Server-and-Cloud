## ⚔️ The Scenario

Cloud environments are powerful—but often misconfigured.

Thunderbyte knows this.

They don’t rely on zero-days.  
They rely on:
- Weak configurations  
- Poor identity management  
- Human oversight  

Cloud Tech exists to find and fix those weaknesses before attackers do.

---

## 🎯 Purpose of This Lab Series

These labs are designed to simulate real-world enterprise environments and explore them from both perspectives:

### ☁️ Cloud Tech (Defender Mindset)
- Secure Active Directory environments  
- Configure DNS correctly  
- Harden systems  
- Detect vulnerabilities  

### ⚡ Thunderbyte (Attacker Mindset)
- Enumerate the network  
- Exploit misconfigurations  
- Move laterally  
- Identify weak points in infrastructure  

---

## 🧠 Why This Matters

Understanding security isn’t just about defense.

👉 You need to think like an attacker to defend like a professional.

These labs demonstrate:
- How environments are built  
- How they are broken  
- How they are secured
